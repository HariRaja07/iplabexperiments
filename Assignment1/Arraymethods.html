SOME:

The Array.some() method checks if any of the elements in an array pass a test (provided as a function).

REVERSE:

The Array.reverse() method reverses the order of the elements in an array.

SORT:

The Array.sort() method sorts the elements of an array.

MAP:

The Array.map() method creates a new array from the results of calling a function for every element.
